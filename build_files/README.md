# 编译文件文件夹

本文件夹包含项目中的所有编译相关文件：

## 文件说明

1. **build_exe.py** - 打包脚本
   - 将音频转录与总结工具打包成exe文件
   - 使用PyInstaller进行打包
   - 支持创建便携式包

2. **音频转录与总结工具.spec** - PyInstaller配置文件
   - 定义了打包的详细配置
   - 包含依赖项、数据文件等设置

3. **build/** - 编译输出目录
   - 包含PyInstaller生成的中间文件
   - 包含编译过程中的临时文件

## 使用方法

### 打包成exe文件

```bash
python build_files/build_exe.py
```

### 直接使用PyInstaller

```bash
pyinstaller build_files/音频转录与总结工具.spec
```

## 注意事项

- 打包前请确保已安装所有必要的依赖
- 打包过程可能需要较长时间，请耐心等待
- 打包完成后，生成的exe文件位于dist目录中
- 如果修改了项目结构或添加了新的依赖，可能需要更新.spec文件

## 输出文件

打包完成后，会生成以下文件：
- `dist/音频转录与总结工具.exe` - 单文件可执行程序
- `dist/音频转录与总结工具_便携版/` - 包含配置文件和目录的便携版
